# Laurensius Haryo Radyobaskoro P - Portfolio

A data-driven Quality Assurance professional and Game Developer portfolio.

## Features
- **Dark Mode**: Seamless toggle between light and dark themes.
* **Interactive Modals**: Detailed project views with smooth animations.
* **Responsive**: Works perfectly on mobile and desktop.
* **Automated Deployment**: Built-in GitHub Actions for GitHub Pages.

## Tech Stack
- React 19
- TypeScript
- Tailwind CSS 4
- Motion (Framer Motion)
- Lucide Icons

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`
